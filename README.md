# Refactory Clone Website

A front-end clone of the Refactory website homepage, built to demonstrate modern web design techniques including responsive layouts, navigation menus with dropdowns, hero carousel banners, and structured content sections.
This project is part of a course assignment for practicing web development fundamentals.

## Key Features

- Responsive UI (Desktop, Tablet & Mobile)

- Navigation bar with hover dropdowns

- Hero section with fullscreen image carousel

- About section with text + imagery side by side

- Featured Programs / Courses section with cards

- Footer with social media icons and contact info


## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid Layout, gradients, transitions, media queries) 
- Font Awesome (for icons)
- JavaScript (for carousel or menu toggle)
